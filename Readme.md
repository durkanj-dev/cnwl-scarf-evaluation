# SCARF Programme Staff Evaluation Survey

## Overview
This repository hosts the front-end code for the CNWL SCARF Programme Evaluation Survey (2026). The survey is designed to assess staff awareness, behavioral change, and perceived value for money following the £600,000 investment in the SCARF cultural framework.

## Transparency & Integrity
To ensure a "tamper-proof" evaluation:
1. **Version Control:** All changes to the survey questions are logged via GitHub commits.
2. **Anonymity:** No IP addresses or identifying PII are sent to the Google Sheets backend.
3. **Open Source:** The logic for data handling is visible in `google-apps-script.js`.

## Technical Setup
- **Front-end:** HTML5/CSS3/JavaScript (WCAG 2.1 AA Compliant).
- **Backend:** Google Apps Script (Web App).
- **Data Store:** Google Sheets.

## Disclaimer
This is an independent evaluation tool intended for internal Trust performance review.
